What is the difference between UI & UX?

UX (User Experience) in a few words: functionality of the product (web application, furniture or a bottle of shampoo!). UX includes all aspects of the end user's interaction with the company, its services, and its products. In other words connects with each other technology, design and business. UI (User Interface) is everything what the user sees: colors, typography, distance between elements and much more!

To meet users' needs you have to think out of the box!

Ask yourself if your product is...

  • useful?
  • easy to use?
  • fun?

To create a product which make our users happy, we need to follow UI & UX rules. Everything matters and there are some certain steps which shouldn't be omitted, e.g. user's research, empathy maps, customer journey maps, creating personas, wireframing and prototyping, usability testing and much more!

The most common roles in UX

  • UX Designer - creates prototypes, wireframes
  • UX/UI Designer - creates prototypes, takes care of visual design
  • UX Researcher - conducts user interviews and usability testing, designs survey
  • Information Architect - creates wireframes, navigation and hierarchy, analyses user research
  • Front-End Developer - writes functional code, uses HTML, CSS, JavaScript
  • UX Copywriter - takes care of the language which is the most convenient for the users
  • Visual/Graphic Designer - creates icons and visual elements for UI, create and applies styles
